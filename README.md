## Installation    
* Clone this repo
```sh
git clone https://github.com/dibenso/Pokedex.git
```     
* Install dependencies
```sh
yarn install
```     
or
```
npm install
```

## Using     
To use the Pokédex app:      
```sh
npm start
```
    
Then check [http://localhost:3000]()
